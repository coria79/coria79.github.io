# Sitio preliminar - Soluciones para Ortopedias

Archivos incluidos:

- `index.html`
- `styles.css`

Los íconos se cargan desde Font Awesome mediante CDN.
La fuente Montserrat se carga desde Google Fonts.

Para verlo:
1. Descomprimir el ZIP.
2. Abrir `index.html` en el navegador.

Nota:
La imagen principal no usa una imagen real todavía. Está simulada con CSS para poder evaluar estructura, colores, espaciado, jerarquía visual y acentos naranjas sin depender de recursos externos.
