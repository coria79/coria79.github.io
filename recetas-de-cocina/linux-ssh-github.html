Receta para conectar GitHub con SSH desde Linux

Introducción:

Como desarrollador será necesario habituarse al entorno Linux. Y una de las tareas necesarias será conectarse a un repositorio remoto como lo puede ser GitHub. Para poder conectarse de forma segura una de las formas de poder hacerlo es por medio de una llave SSH.

Objetivo:

Mostrar los pasos necesarios para poder realizar de forma exitosa la conexión con un repositorio remoto por medio de una llave ssh para autentificar al usuario.

Ingredientes:

- Ubuntu Linux
- Terminal
- Navegador
- Cuenta activa en GitHub
Tiempo de cocción:

- 10 a 15 minutos.

Instrucciones:

- Iniciar Terminal.
- Iniciar Navegador.
- En el navegador será necesario iniciar sesión en la cuenta de GitHub.
- En la Terminal será necesario correr los siguientes comandos.

Actualizar el sistema operativo.
sudo apt update && sudo apt upgrade -y

Comprobar si Git está instalado.
git --version

Si no está instalado ejecutar el siguiente comando para instalarlo.
sudo apt install git

Verificar de estar en el “home” del usuario
pwd

Ingresa a la carpeta oculta .ssh
cd .ssh

Generar la clave SSH.
ssh-keygen

Solicitará ingresar el nombre del archivo.
Al estar parados en la misma carpeta de las llaves ssh no será necesario ingresar la ruta.
id_ed25519_test
Presionar Enter

Solicitará una passphrase para aumentar el nivel de seguridad.
En este caso no será necesario, solo presionar Enter
Volverá a solicitar la confirmación de la passphrase, solo presionar Enter

Se mostrará el resultado de la creación de la llave SSH.
Your identification has been saved in /home/tu_usuario/.ssh/id_ed25519_test
Your public key has been saved in /home/tu_usuario/.ssh/id_ed25519_test.pub

Verificar que los archivos se encuentran en la carpeta
id_ed25519_test ← clave privada (no se comparte)
id_ed25519_test.pub ← clave pública (se copia a GitHub)

Mostrar el contenido de la clave pública.
cat id_ed25519_test.pub

Copiar toda la línea que se muestra para llevarlo a la página web de GitHub.
En el navegador, entra a tu cuenta GitHub.
Haz clic en tu avatar → “Settings” → “SSH and GPG keys”.
Pulsar New SSH key.
Title: Clave Linux
Key: pegar el contenido copiado, solo debe ser la clave pública.
Clic en Add SSH key.


En la terminal ejecutar los siguientes comandos.
ssh -T git@github.com

Cuando consulta sobre si confiar en el host teclear "yes" y presionar Enter.
Si todo salió bien se verá lo siguiente.
Hi <tu_usuario>! You've successfully authenticated

Elige una carpeta donde trabajar, por ejemplo:
cd Documentos
Clonar el repositorio con:
git clone git@github.com:<TU_USUARIO>/<TU_REPO>.git

Comentarios finales:

La clave privada solo debe estar en la máquina local, no se debe compartir con nadie. La generación de la llave SSH permite conectarse de forma segura con el repositorio.
