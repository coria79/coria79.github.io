Django Framework on Ubuntu from Windows with Virtual Machines.

Introduction:

It's preferable to develop in a Linux environment. Most likely, your computer's operating system is Windows. To avoid formatting the computer and installing a Linux distribution. Or to avoid creating a new partition with a file system different from that of Windows, this recipe explains how to access Linux without giving up Windows on the computer.

Objective:

From Windows and through a virtual machine, be able to install everything needed for Django development in a Linux environment. This recipe covers all the necessary steps.

It is divided into four sections. By completing all sections, you will have everything you need to start development.

Recipe to download and install VMware Workstation Player on Windows:



Ingredients:

- Web Browser.
- VMware Workstation Player (This recipe is based on version 17).

- Installation file size: 577 Mb (approx.)
- Installed application size: 841 Mb (approx.)

Cooking Time:

- 10 to 15 minutes.

Prerequisites:

- Verify that the computer meets the minimum or recommended requirements (preferably recommended: dual-core processor with four threads and 8Gb RAM).
- Windows operating system is up-to-date.
- No pending updates to be installed.
- No other applications running, close all unnecessary windows.

Download:

- From the web browser, visit the official VMware download page.
- Scroll down to find the "Download Trial Version" for the current version of VMware Workstation Player for Windows and download it.

Installation:

- Double-click on the installation file (*.exe) to start the installation wizard (next).
- Accept the license terms (next).
- Optional: Add keyboard drivers.
- Leave "Add VMware Workstation console tools to the system PATH" checked (next).
- Leave "Check for product updates on startup" checked.
- Optional: uncheck "Join the user experience improvement program" (next).
- Shortcuts can be left at the default settings (next).
- "Confirm" to start the installation.
- Notification appears that the installation is complete (finish).
- Optional: Restart the system (recommended).

Initial Configuration:

- Open VMware Workstation Player from the start menu (or from the desktop icon).
- Choose "Use VMware Workstation Player for free" (continue).
- "Finish," and the main VMware Workstation Player window will open.


Recipe for downloading, installing, and configuring VMware Workstation Player on a Windows system is complete.

Recipe to download Ubuntu Linux ISO image:



Ingredients:

- Web Browser.
- ISO file size: 4.69 Gb (approx.).

Cooking Time:

- 20 to 30 minutes (depends on the ISO download speed).

Download:

- From the web browser, visit the official Ubuntu download page.
- Scroll down to find the current version of Ubuntu to download. It is recommended to choose the LTS (Long Term Support) version.
- Click the "Download" button to start the download process.
- Optional: Scroll down to the "verify your download" section using the SHA-256 hash of the ISO.

Recipe for downloading Ubuntu Linux ISO image in VMware Workstation Player on Windows system is complete.

Recipe to create a virtual machine of Ubuntu Linux:



Ingredients:

- VMware Workstation Player.
- Ubuntu Linux ISO image.

- Minimum installation of Ubuntu takes up 14.5 Gb (approx.)

Cooking Time:

- 15 to 20 minutes.

Instructions:

- Open VMware Workstation Player.
- Choose the option "Create a New Virtual Machine."
- The "New Virtual Machine Wizard" window opens.
- Click "Browse..." to locate the Ubuntu Linux ISO file.
- A window opens to navigate through folders to find the Linux Ubuntu .ISO file.
- Select the .ISO file (Open).
- Click the "Next" button.
- Fill in the fields with user data (Next).
- Name the virtual machine and select the destination for the new virtual machine. It can be left as the default (Next).
- Define the size of the hard disk. It is recommended to use 40Gb.
- Check "Split into multiple files" (Next).
- To increase the RAM size, select the "Edit Hardware" button.
- Modify the RAM size value to 8192 (Close).
- Check the option "Power on this virtual machine after creation" (Finish).
- The Ubuntu Linux installation process starts.
- Keyboard Layout. Press the "Detect Keyboard Layout" button. It will ask to press some keys for correct detection (Continue).
- Choose the "Minimal Installation" option.
- Check both options in "Other Options" (Continue).
- Installation Type. Leave selected as shown by default. Erase Disk and Install Ubuntu. Leave everything as it is (Install Now).
- A new popup window appears showing "Changes to disk" (Continue).
- Choose the time zone (Continue).
- Enter Full Name, Username, Password, Confirm Password.
- Check the option "Require my password to log in" (Continue).
- The installation process of the operating system is displayed.
- At the end of the installation, a new popup window opens (Restart Now).
- Log in.
- A popup window appears with final installation details.
- Connect with online accounts (Skip).
- Enable Ubuntu Pro. Check the option "Skip for now" (Next).
- Help improve Ubuntu. Check the option "No, do not send system information" (Next).
- Privacy. Leave the option "Location Services" deactivated (Next).
- Ready to continue (Done).
- A popup window may appear asking to install updates (Install Now).
- Enter the user password (Authenticate).
- At the end of the update, it will ask to restart the system (Restart Now).

Installation of VM-Tools:

- Log in.
- Open "Terminal".
- Enter the command: sudo apt install open-vm-tools (enter).
- Enter the user password.

Recipe for creating Ubuntu Linux Virtual Machine with VMware Workstation Player is complete.

Recipe to configure the development environment for Django on Ubuntu Linux:



Ingredients:

- PyCharm.
- DBeaver.
- Python.
- Pip.
- Virtual Environment.
- Django.

- Installed PyCharm application size: 800 Mb (approx.).
- Installed DBeaver application size: 259 Mb (approx.).
- Installed Pip application size: 221 Mb (approx.).
- Installed Virtual Environment application size: 4 Mb (approx.).
- Installed Django application size: 8 Mb (approx.).

Cooking Time:

- 10 to 15 minutes.

Prerequisites:

- Ubuntu Software application.

Install PyCharm:

PyCharm is the IDE (Interface Developer Environment).
- Launch the "Ubuntu Software" application.
- From the search, perform a PyCharm search.
- Results of the search will be automatically displayed.
- Select "PyCharm Community Edition".
- Details of "Pycharm Community" will be displayed (Install).
- Once the installation is complete, close Ubuntu Software.
- A popup window will appear asking to enter the user password (Authenticate).
- Start PyCharm to validate that it installed correctly.
- Select "I confirm that I have read and accept the terms of this user agreement" (Continue).
- Share Information (Do Not Send).
- The application will start for the first time, showing the Welcome screen.

Install DBeaver:

DBeaver is the database manager.
- Launch the "Ubuntu Software" application.
- From the search, perform a DBeaver search.
- Results of the search will be automatically displayed.
- Select "dbeaver Universal Database Manager and SQL Client".
- Details of "DBeaver" will be displayed (Install).
- A popup window will appear asking to enter the user password (Authenticate).
- Once the installation is complete, close Ubuntu Software.
- Start DBeaver to validate that it installed correctly.
- A popup window will appear asking to create a sample database (No).
- The database wizard will be displayed.
- A "Send Statistical Information" window may appear (Do Not Send).

Python Installation:

Python is already installed with Ubuntu.
- To verify the current installed version of Python.
- Open terminal.
- Enter the following command: python3 --version (enter).

Pip Installation:

Pip is the package manager needed to install Django.
- Open terminal.
- Enter the following command: sudo apt install python3-pip (enter).
- Enter the user password.
- It will request confirmation to install (Y).

Virtual Environment Installation:

The virtual environment is necessary to run a Django project.
- Open terminal.
- Enter the following command: sudo apt install python3 virtualenv (enter).
- Enter the user password.
- It will request confirmation to install (Y).

The recipe to configure development environment for Django on Ubuntu Linux has been completed.
